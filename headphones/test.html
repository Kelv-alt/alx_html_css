Responsive Hamburger Menu

This menu transforms into a mobile-friendly hamburger menu at 480px screen width. Click the hamburger icon to see the animation!

How It Works

The hamburger menu uses CSS media queries to activate at 480px screen width. JavaScript toggles the "active" class on the navigation menu, which triggers the CSS animation.

On screens larger than 480px, the navigation menu is always visible. On smaller screens, the menu collapses and can be toggled with the hamburger button.
